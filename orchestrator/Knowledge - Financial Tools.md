# Financial Tools

## Overview of SSC Project Management Tools

| Tool | Purpose | Authoritative For |
|------|---------|-------------------|
| **EPS** | Project tracking and reporting | Project data, risks, issues, lessons learned |
| **SIGMA** | Financial system | Actual costs, IO codes |
| **PICT** | Multi-year project costs | Project cost estimates |
| **PCW** | Fiscal year financial tracker | In-year forecasts |
| **ePET** | Customer pricing | Recovery agreements |
| **P2P** | Procurement | Procurement requests |
| **ONYX** | Service line work | Work assignments |
| **Microsoft Project Online** | Scheduling | Project schedules |

---

## Enterprise Portfolio System (EPS)

### Purpose
SSC's Project Management Information System (PMIS) - official departmental system of record for project information.

### Key Uses
- Project tracking and reporting
- Risk management
- Issue tracking
- Lessons learned documentation
- Artifact storage (final approved versions)

### Responsibilities
- Project Managers are accountable for keeping EPS up to date
- Final project documentation must be stored in EPS
- Also store in GCDOCS (corporate IM repository)

### Data Model
- Integrates with all other project tools
- Supports portfolio-level reporting

---

## SIGMA Financial System

### Purpose
Authoritative source for all project costs, built on SAP technology.

### Key Features
- **Internal Order (IO)**: Assigned after Gate 0, tracks actual costs
- **CATS (Cross-Application Time Sheets)**: Monthly time reporting for all project resources
- **BPC (Branch Planning and Consolidation)**: Yearly forecasts (Periods 3, 6, 9)

### Responsibilities
- All resources must enter time in CATS monthly
- Project actuals tracked via IO number
- Used for Financial Situation Report (FSR)

---

## PICT - Project and Investment Costing Tool

### Purpose
Official tool for documenting multi-year project costs. Ensures uniform costing across projects.

### Key Features
- Evergreen document - updated continuously
- Expresses costs in terms of deliverables and timelines
- Cost by gate, stage, and deliverable

### When to Use
- Initial cost estimate at Gate 0
- Refinement at each subsequent gate
- Track costs vs. baseline

### Support
- CMAs provide continual guidance
- Contact CMA at project outset

---

## PCW - Project Control Workbook

### Purpose
Standardized Excel template for capturing, managing, and tracking detailed financial data by fiscal year.

### Key Features
- Tracks budget, actual, and forecast
- Integrates with SIGMA and ePET data
- Monthly updates required

### When to Use
- In-fiscal year financial tracking
- Monthly variance reporting
- Forecast management

### Update Frequency
- Update regularly throughout each month
- Explain month-to-month variances in forecasts

---

## ePET - Enterprise Price Estimation Tool

### Purpose
Generate customer project cost estimates and Recovery Agreements.

### Key Features
- **Standard Pricing Module**: For services
- **Custom Pricing Module**: REQUIRED for PRAs (Project Recovery Agreements)

### When to Use
- Customer-Led projects (partially or wholly funded by VNR)
- Creating Project Recovery Agreements
- Any custom pricing scenario

### Important
- Always use Custom Pricing for PRAs
- Standard pricing assumptions are not appropriate for project costing
- Supporting evidence required for all cost elements

---

## P2P - Procure-to-Pay Portal

### Purpose
Electronic collaborative platform for procurement process.

### When to Use
- After Gate 3 approval
- Submit procurement requisitions
- Track procurement lifecycle
- Order placement, delivery, payment

### Access
- Exclusive to SSC employees
- Post-Gate 3 only

---

## ONYX

### Purpose
Information Technology Service Management (ITSM) tool for service requests and incidents.

### Key Features
- Self-service portal
- Automated workflows
- Task management
- Reporting capabilities

### For Projects
- Request user groups for projects
- Create personalized workflows
- Assign tasks to service lines

### Training
- ONYX training and support SharePoint available

---

## Microsoft Project Online

### Purpose
Cloud-based project management for planning, managing, and tracking schedules.

### Key Features
- **Gantt Charts**: Visual timeline
- **Task Dependencies**: Finish-to-start, start-to-start, etc.
- **Resource Management**: Assign people/equipment
- **Real-Time Collaboration**: Teams integration

### For Projects
- Create Work Breakdown Structure (WBS)
- Establish timeline with milestones
- Assign resources
- Track progress against baseline

---

## CATS Time Reporting

### What is CATS?
Cross-Application Time Sheets within SIGMA for reporting time spent on projects.

### Requirements
- **All resources** involved in a project must enter time monthly
- Monthly reporting mandatory
